## Tensura Compat: FancyMenu examples

This folder contains several example files which use placeholders and requirements from this mod. These files were
created in-game using FancyMenu and SpiffyHUD, so both mods will be required alongside Tensura: Reincarnated in order to
see them render fully and correctly. Just download and drop the folder into the root of a modpack containing all the
prerequisite mods in order to view the overlays in action.
