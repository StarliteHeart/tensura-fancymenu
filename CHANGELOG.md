## Changelog

***

### v0.1.0 Beta (Released July 20, 2026)

#### Initial beta release!

- Added 40 new Placeholders and 9 new Requirements to FancyMenu that provide support for Tensura-related values.
    - New Placeholders
        - Player Race Name
        - Player Tracked Evolution Name
        - Current % Player Evolution Progress
        - Player Alignment
        - Current, Max and Current % Player Magicules
        - Current, Max and Current % Player Aura
        - Current, Max and Current % Player Existence
        - Current, Max and Current % Player Spiritual Health
        - Current, Max and Current % Mount Spiritual Health
        - Current and Current % Player Health Severance
        - Player Multilayer Barrier
        - Player Soul Points
        - Player Analysis Mode Level
        - Player Presence Sense Level
        - Player Dodge Cooldown
        - Player Magicule Poison Resistance
        - Ability Active Preset
        - Ability Slot Skill Name
        - Ability Slot Skill Icon (Resource Location)
        - Ability Slot Mode Name
        - Ability Slot Mode Index
        - Current, Max and Current % Ability Slot Learning
        - Current, Max and Current % Ability Slot Mastery
        - Ability Slot Cooldown
        - Area Magicules
    - New Requirements
        - Is Player Demon Lord Seed
        - Is Player True Demon Lord
        - Is Player Hero Egg
        - Is Player True Hero
        - Is Player In Spiritual Form
        - Is Ability Slot Filled
        - Is Ability Slot Learned
        - Is Ability Slot Mastered
        - Is Ability Slot On Cooldown
- Patched an issue between FancyMenu and Tensura: Reincarnated where having any amount of menu blur as set through
  Tensura's config would cause screens to not render correctly in larger modpacks.
