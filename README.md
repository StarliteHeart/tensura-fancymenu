# Tensura Compat: FancyMenu

A compatibility bridge between Tensura: Reincarnated and FancyMenu. This mod introduces 40 new Placeholders and 9 new
Requirements to help modpack developers or the casual player construct Tensura-based interfaces as they desire.

It also fixes an existing issue between Tensura: Reincarnated and FancyMenu where Tensura screens may sometimes
disappear or render incorrectly in larger modpacks with FancyMenu.

You can find this mod on both [CurseForge](https://www.curseforge.com/minecraft/mc-mods/tensura-compat-fancymenu)
and [Modrinth](https://modrinth.com/mod/tensura-compat-fancymenu).

This addon is currently in ongoing beta development! If you have any recommendations for new placeholders, requirements
or elements that you'd like to see added for FancyMenu support, you may create a
new [Feature Request](https://github.com/StarliteHeart/tensura-fancymenu/issues/new?template=feature-request.md) to
detail the addition. Likewise, if you discover any bugs or other issues, please file
a [Bug Report](https://github.com/StarliteHeart/tensura-fancymenu/issues/new?template=bug-report.md) instead. Please
check the issues page first to make sure that an existing ticket for the feature or bug does not already exist before
creating a new ticket.
