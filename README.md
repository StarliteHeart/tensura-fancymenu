# Tensura Compat: FancyMenu

A compatibility bridge between Tensura: Reincarnated and FancyMenu. This mod introduces 40 new Placeholders and 9 new
Requirements to help modpack developers or the casual player construct Tensura-based interfaces as they desire.

It also fixes an existing issue between Tensura: Reincarnated and FancyMenu where Tensura screens may sometimes
disappear or render incorrectly in larger modpacks with FancyMenu, while also switching the mod to use Minecraft's
built-in background blur setting instead of its own for consistency. (If you have Iris Shaders installed, this will also
fix the same issue with its shader selection screen in-world.)

You can find this mod on both [CurseForge]() and [Modrinth]()!
